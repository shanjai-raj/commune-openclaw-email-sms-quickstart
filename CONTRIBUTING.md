# Contributing

Thanks for your interest in contributing to commune-openclaw-email-quickstart!

## Ways to contribute

- **Report bugs** — Open an issue describing what broke and how to reproduce it
- **Improve skill files** — The SKILL.md files are the heart of this project. Better prompts = better agent behavior
- **Add examples** — New example prompts, use cases, or configuration snippets
- **Improve docs** — Clearer setup instructions, better troubleshooting tips

## Submitting a PR

1. Fork the repo
2. Make your changes
3. Test the skill by installing it locally (`bash install.sh`)
4. Open a PR with a clear description of what changed and why

## Questions?

Open an issue or email hello@commune.email.
